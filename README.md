# Nanoc::Graphviz

Nanoc Graphviz filter plugin.

## Installation

Add this line to your application's Gemfile:

    gem 'nanoc-graphviz'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install nanoc-graphviz
